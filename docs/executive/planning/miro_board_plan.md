## 📋 ИНСТРУКЦИЯ: КАК СОЗДАТЬ В MIRO

### ВАРИАНТ 1: Ручное создание (рекомендуется)

1. **Откройте Miro** → Новая доска
2. **Создайте 5 колонок** (используйте фреймы):
   - Колонка 1: КОНТЕКСТ
   - Колонка 2: БЛОК 1
   - Колонка 3: БЛОК 2
   - Колонка 4: БЛОК 3
   - Колонка 5: РЕЗУЛЬТАТЫ

3. **Для каждой карточки**:
   - Используйте стикер (Sticky Note) или карточку (Card)
   - Копируйте текст из этого документа
   - Назначайте цвет по приоритету
   - Добавляйте иконки (в Miro есть встроенные)

4. **Добавьте связи**:
   - Используйте стрелки между задачами
   - Покажите зависимости (Блок 1 → Блок 2 → Блок 3)

5. **Назначьте ответственных**:
   - В Miro можно добавить аватар члена команды
   - Перетащите аватар Нини/Насти/Ивана на карточки

### ВАРИАНТ 2: Импорт через CSV (быстрее)

1. Использовать файл `miro_import.csv` (создам отдельно)
2. В Miro: More tools → Import data → Upload CSV
3. Настроить маппинг колонок
4. Импортировать

Создам готовый шаблон (если есть Miro аккаунт с возможностью экспорта).
