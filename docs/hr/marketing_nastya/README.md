# HR / Маркетинг / Настя

**Цель папки:** дать маркетологу Анастасии (и HR) готовые рамки мышления, промты и материалы для годовой стратегии **«300 LaSpecialista в год»** и обучения команды.

**Базовый контекст:** основной канал продаж — магазины в молах + кофейни при точках (стенды с техникой). Инструменты: визуальная реклама, рассылки, акции, оформление точек, работа продавцов.

## Структура

- `README.md` — вы сейчас здесь. Кратко объясняет структуру.
- `PROMPTS_FOR_ANASTASIA_300_LA_SPECIALISTA.md` — стратегические промты для Насти (в том числе для работы с AI) по постановке целей, выбору фокуса и планированию года+.
- `PROMPTS_FOR_STAFF_LA_SPECIALISTA_SALES.md` — вопросы, мини-упражнения и истории для работы с командой продаж (в стиле «история + вопрос»), чтобы сотрудники сами доходили до нужных выводов.

## Как использовать

- **Александр / директор**: задаёт стратегическое направление, уточняет формулировки целей и метрик, возвращается сюда, чтобы напомнить себе рамки.
- **Настя / маркетолог**: регулярно работает с промтами, формирует стратегию, помесячный план, брифы на кампании и материалы для обучения продавцов.
- **Тимур / менеджер и HR**: забирают блоки из `PROMPTS_FOR_STAFF_LA_SPECIALISTA_SALES.md` для встреч с командой, тренингов и разборов ситуаций.

Общий принцип тот же, что и в документах для Тимура: **мы не даём готовые ответы, мы задаём сильные вопросы и даём рамку, внутри которой команда принимает решения и растёт.**
