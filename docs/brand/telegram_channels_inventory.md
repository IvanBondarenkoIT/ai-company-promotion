# Реестр Telegram-каналов DimKava (сверка)

**Назначение:** единая таблица для **ручной** сверки живых чатов в Telegram с документами и с выгрузкой из Excel.

**Источник данных (архив анализа):** [data/input/TG Chanels Analize.xlsx](../../data/input/TG%20Chanels%20Analize.xlsx) — имя файла с опечаткой *Chanels*; при переименовании файла обновите ссылку здесь и в [telegram_channels_recommendations.md](telegram_channels_recommendations.md).

**Канон именования:** [telegram_channels_naming_standard.md](telegram_channels_naming_standard.md), **11 логических каналов** — [telegram_channels_recommendations.md](telegram_channels_recommendations.md).

---

## Процедура сверки

1. Открыть выгрузку / Excel и список чатов в Telegram (аккаунт администратора).
2. Для каждой сущности заполнить строку ниже: **фактическое имя**, **тип** (канал / группа / бот), **@username** если есть, **совпадение с каноном** (да / переименовать / слить).
3. После переименования занести **новую** ссылку `https://t.me/...` (username меняется не всегда вместе с title — проверить в настройках чата).
4. Указать дату сверки внизу файла и при необходимости обновить дату в [telegram_channels_quick_reference.md](telegram_channels_quick_reference.md).

---

## Матрица перехода (шаблон)

| Старое отображаемое имя (как в TG) | Тип | Старый @username / ссылка | Канон (новое имя) | Новый @username / ссылка | Дата | Ответственный |
|-------------------------------------|-----|---------------------------|-------------------|--------------------------|------|---------------|
| *пример: склад заказы…* | канал | | `DK Warehouse \| Orders` | | | |
| … | | | | | | |

Дубли и «лишние» каналы на другом языке помечать в колонке **Канон** как «слить в `DK …`».
